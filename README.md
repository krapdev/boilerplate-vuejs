# boilerplate-vuejs

> Boilerplate for Vue.js application

## tech stack

``` bash
> Vue.js 2.x
# https://vuejs.org/v2/guide/

> based on webpack template
# [template guide] https://github.com/vuejs-templates/webpack
# [vue-loader guide] http://vuejs.github.io/vue-loader

> eslint configuration based on Vue.js strong-recommended styles guide and airbnb
# https://vuejs.org/v2/style-guide/#Priority-B-Rules-Strongly-Recommended-Improving-Readability

> jest and vue-test-utils for unit test
# https://facebook.github.io/jest/
# https://vue-test-utils.vuejs.org/fr/

> vuex for state management pattern
# https://vuex.vuejs.org/fr/

> vue-router
# https://router.vuejs.org/fr/

> Axios for http fetching
# https://github.com/axios/axios
```

## npm commands

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# check syntaxics best practices from vuejs-strong-recommended and airbnb-base
npm run lint

# run all tests (jest and @vue/test-utils for component testing)
npm run test

# run all tests and report coverage
npm run coverage
```
